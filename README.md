# Map Pins

> **Note:** WIP. This is a learn-by-doing project.

Map Pins is a realtime social map-pinning web app. This project uses the following technologies:

**Frontend**

- React
- GraphQL (Apollo Client)
- Material-UI

**Backend**

- Node
- GraphQL (Apollo Server)
- MongoDB (Mongoose)

**Dev Tools**

- babel-node
- ESlint

**Services Used**

- Google OAuth2
- MongDB Atlas
- Mapbox
- Cloudinary
